AI 圖書館員

設定 OpenAI API Key 以啟用 AI 圖書館員聊天功能。您的 Key 僅儲存在本機瀏覽器中，不會上傳至任何伺服器。

💡 前往 platform.openai.com 取得您的 API Key。建議使用 gpt-4o-mini 模型，兼顧效能與成本。

Google Drive 同步

連接您的 Google Drive，自動匯入儲存的書本檔案。

💡 需要在 Google Cloud Console 建立 OAuth2 Client ID，並將 https://liyanongithub.github.io 加入授權來源。
同步時自動從試算表讀取書名、作者、分類、封面等欄位。試算表需有 Path 欄（放 Drive 檔案連結）。
